`디렉토리 구조`
```
src
├─apis          // 서비스 제공 기능 API구조
├─assets        // font와 image 정의
├─components    // 서비스에 사용되는 component
│  ├─common
│  ├─login
│  └─signup
├─pages         // 만들어둔 component들의 feature들을 모아 페이지 구성
│  ├─login
│  └─signup
├─store         // 상태관리 정의
└─styles        // style 정의
```
